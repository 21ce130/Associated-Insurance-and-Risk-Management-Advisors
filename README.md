# Associated Risk Engine

**Commercial Insurance Risk Assessment & Carrier Comparison Platform**

An AI-powered risk scoring tool built for the insurance industry. Features multi-dimensional risk analysis, Monte Carlo premium simulation, coverage gap assessment, carrier matching, and regulatory compliance flagging — tailored to Associated Insurance & Risk Management Advisors' business lines and carrier partnerships.

## Features

- **12 Industry Verticals** with 60 sub-classifications
- **14 Coverage Lines** with priority scoring and premium estimation
- **12 Carrier Partners** with match scoring by industry fit, coverage alignment, and financial strength
- **Monte Carlo Premium Simulation** (1,000 scenarios) with percentile ranges
- **5-Year Loss Ratio Projection** with premium forecasting
- **50-State Regulatory Data** including WC modifiers and complexity scores
- **Compliance Flagging Engine** (DOT/FMCSA, HIPAA, ACA, state-specific)
- **Risk Mitigation Action Plan** with ROI estimates

## Tech Stack

- React 18 + Vite
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```
